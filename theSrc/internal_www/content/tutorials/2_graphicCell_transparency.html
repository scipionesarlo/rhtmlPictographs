There are two syntax options: object or string shorthand 

{ "variableImage": { "type": "circle", "color":"blue", "opacity":"0.5" } }
{ "variableImage": "circle:blue:opacity=0.5" }

Multi Image Graphic Cells: proportion and transparency

The opacity applies to all images in the graphic cell. If you want different opacity values, create a table of multiple graphic cells (see table tutorial for table construciton syntax)

{ "numRows": 5, "numImages": 20, "proportion": 0.975, "variableImage": "circle:fromleft:blue:opacity=0.1" }

Circles with transparency

{ "variableImage": "circle:blue" }
{ "variableImage": "circle:blue:opacity=0" }
{ "variableImage": "circle:blue:opacity=0.33" }
{ "variableImage": "circle:blue:opacity=0.66" }
{ "variableImage": "circle:blue:opacity=1" }
